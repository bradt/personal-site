Rwyn

Reveal what you need

Rwyn was designed with usability in mind and created as an alternative to drop-down menus. It is composed of XHTML 1.0, CSS 2.0 and uses simple cross-browser compliant Javascript to manipulate the DOM.

Compatibility:

  • MS IE 5+
  • Firefox 1.0.7+
Click here to post your compatibilty testing results.

To get Rwyn, simply view source or download this page. The Javascript and CSS is embedded in the page for easy viewing and downloading.
